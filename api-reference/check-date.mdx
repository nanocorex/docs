---
title: "Check Date"
api: "GET /api/check-date/{countryCode}/{date}"
description: "Checks whether a specific date is a holiday"
---

Checks if a specific date is a holiday, weekend, or working day for a given country.

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `countryCode` | string | Yes | ISO 3166-1 alpha-2 country code (e.g., "TR", "US") |
| `date` | string | Yes | Date in YYYY-MM-DD format (e.g., "2025-04-23") |

## Response

Returns information about whether the date is a holiday or working day.

### Example Response

```json
{
  "date": "2025-04-23",
  "country": "TR",
  "isHoliday": true,
  "isWeekend": false,
  "isWorkingDay": false,
  "holiday": {
    "name": "National Sovereignty and Children's Day",
    "type": "public"
  }
}
```

<RequestExample>
```bash cURL
curl -X GET "https://api.holidayapi.dev/api/check-date/TR/2025-04-23" \
  -H "X-API-KEY: your_api_key"
```
</RequestExample>
