# API Reference

Holiday API is a RESTful API that provides access to official holidays and calendar data for countries worldwide.

## Base URL

```
https://api.holidayapi.dev
```

## Authentication

All API requests require sending your API key via the `X-API-KEY` header.

```bash
curl -X GET "https://api.holidayapi.dev/api/countries" \
  -H "X-API-KEY: your_api_key_here"
```

## Response Format

All responses are returned in JSON format. Successful requests return with a `200 OK` status code, while errors include appropriate HTTP status codes with detailed error messages.

## Rate Limits

| Plan | Requests / Month |
