# API Reference

Holiday API, dünya genelindeki ülkelerin resmi tatil günlerine ve takvim verilerine erişim sağlayan RESTful bir API'dir.

## Base URL

```
https://api.holidayapi.dev
```

## Authentication

Tüm API istekleri için `X-API-KEY` header'ı ile API anahtarınızı göndermeniz gerekmektedir.

```bash
curl -X GET "https://api.holidayapi.dev/api/countries" \
  -H "X-API-KEY: your_api_key_here"
```

## Response Format

Tüm yanıtlar JSON formatında döner. Başarılı istekler `200 OK` status kodu ile dönerken, hatalar uygun HTTP status kodları ile birlikte detaylı hata mesajları içerir.

## Rate Limits

| Plan | Requests / Month |
|------|------------------|
| Free | 1,000 |
| Starter | 10,000 |
| Pro | 100,000 |
| Enterprise | Unlimited |
