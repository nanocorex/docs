Bu rehberde, Holiday API kullanarak desteklenen ülkelerin listesini nasıl çekeceğinizi göreceğiz.

## Ön Hazırlık

İstek yapabilmek için [Kimlik Doğrulama](/authentication) sayfasında anlatıldığı gibi bir API Anahtarına ihtiyacınız var.

## İlk İstek: Ülke Listesi

Aşağıdaki örnekte tüm desteklenen ülkeleri listelemek için `/api/countries` endpoint'ine istek atıyoruz.

<CodeGroup>

```bash cURL
curl -X GET "http://31.220.83.147/api/countries" \
     -H "X-API-KEY: YOUR_API_KEY"
```

```javascript JavaScript
fetch('http://31.220.83.147/api/countries', {
  method: 'GET',
  headers: {
    'X-API-KEY': 'YOUR_API_KEY'
  }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
```

```python Python
import requests

url = "http://31.220.83.147/api/countries"
headers = {
    "X-API-KEY": "YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
print(response.json())
```

</CodeGroup>

## Yanıt Örneği

API başarılı bir istek durumunda JSON formatında ülke listesi dönecektir:

```json
[
  {
    "code": "TR",
    "name": "Turkey"
  },
  {
    "code": "US",
    "name": "United States"
  },
  ...
]
```
