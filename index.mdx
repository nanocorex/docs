# Welcome to Holiday API

**Holiday API** is a RESTful service that provides access to official holidays, working hours, and special days for countries worldwide. It enables developers to easily manage calendar integrations, holiday planning applications, and business workflows.

## Features

- **Official Holidays:** Historical and future official holiday data for 100+ countries.
- **Working Days:** Query whether a specific date or range is a business day.
- **Calendar Data:** Detailed calendar outputs in JSON format.
- **High Performance:** Fast response times and 99.9% uptime.

## Use Cases

- **HR Software:** Working day detection for payroll calculations and leave management.
- **E-Commerce:** Excluding holidays when calculating estimated delivery dates.
- **Calendar Apps:** Automatically processing holiday data into user calendars.
- **Travel Planning:** Preview holiday dates for your destination country.
