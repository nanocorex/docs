To use the Holiday API, you must send a valid **API Key** with every request. This key enables your account authorization and usage quota tracking.

## How to Get an API Key?

1. Log in to the **Dashboard**.
2. Navigate to the **API Keys** menu.
3. Click the **Create New** button to generate your unique key.

<Warning>Never share your API key (`sk_...`) with anyone and do not expose it in client-side (browser) code.</Warning>

## Usage

Send your API key in the HTTP request headers using the **`X-API-KEY`** parameter.

```bash
curl -X GET "https://api.holidayapi.dev/api/countries" \
  -H "X-API-KEY: YOUR_API_KEY"
```

## Authentication Errors

If your API key is missing, invalid, or expired, the API will return a **401 Unauthorized** error.

```json
{
  "status": 401,
  "title": "Unauthorized",
  "detail": "API key is invalid or missing."
}
```
